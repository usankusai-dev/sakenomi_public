# sakenomi

このリポジトリーは「日本全国酒飲み音頭ジェネレーター」の公開リポジトリーとなります。

詳細は下記をご参照ください。

```
https://qiita.com/usankusai/private/0e5e956e61943837f383
```

ローカルでのデプロイ手順は以下のとおりです。

(1)：Node.js をインストールする

```
https://nodejs.org/ja/
```

(2)：リポジトリーをローカルの適当なディレクトリにクローンする

(3)：(2)のディレクトリに移動し、下記のコマンドを実行する。

```
npm install
npm run serve
```
